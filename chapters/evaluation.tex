\chapter{性能测试}
我们在运行Windows Server 2012 R2的戴尔R720服务器上评估ClickNP的性能。
FPGA通过以太网端口与机架顶(top-of-rack, ToR)的戴尔S6000交换机\cite{s6000}相连。

实验结果表明，RoCE能够有效利用网卡的通信带宽，实现比PCIe接口更高的传输速率以及更低的延迟。
使用RoCE替代PCIe通信能够提升FPGA的存储性能。
